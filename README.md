# Patikadev-Insertion

1. Insertion Sort Aşamaları:

[22, 27, 16, 2, 18, 6] 

Adımlar:
1. [22, 27, 16, 2, 18, 6]
2. [16, 22, 27, 2, 18, 6]
3. [2, 16, 22, 27, 18, 6]
4. [2, 16, 18, 22, 27, 6]
5. [2, 6, 16, 18, 22, 27]

Sonuç: [2, 6, 16, 18, 22, 27]

---

2. Big-O Gösterimi:
- Best Case: O(n)
- Average Case: O(n^2)
- Worst Case: O(n^2)

- Average Case: Aradığımız sayı ortada.
